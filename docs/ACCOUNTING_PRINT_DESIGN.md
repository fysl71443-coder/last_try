# مبادئ التصميم المحاسبي للطباعة

## 1️⃣ مبادئ الطباعة الاحترافية (محاسبيًا)

### ما يجب أن يشعر به القارئ
- **رسمي** — مناسب للمراسلات والملفات الرسمية
- **واضح** — سهل القراءة والمراجعة
- **قابل للأرشفة** — يطبَع ويُحفظ دون فقدان وضوح
- **مناسب للمراجعين والمدققين** — جداول منظمة وأرقام قابلة للمراجعة

### ما نتجنبه
- ألوان قوية أو زاهية
- ازدحام النص أو الجداول
- خطوط افتراضية (Times / Arial) — نفضّل خطوط عربية واضحة

## 2️⃣ الهوية البصرية المقترحة

| العنصر | الاختيار |
|--------|----------|
| الخط | Cairo / Tajawal |
| اللون الأساسي | كحلي داكن `#1f2937` |
| لون ثانوي | رمادي هادئ `#6b7280` |
| خلفية | أبيض `#fff` |
| الجدول | حدود خفيفة `#d1d5db` + صفوف متبادلة `#fafafa` |
| التذييل (tfoot) | خلفية `#e5e7eb`، خط عريض |

## 4️⃣ الملفات الجاهزة للاستخدام

| الملف | الاستخدام |
|-------|-----------|
| `templates/partials/accounting_print_styles.html` | تضمينه في أي قالب طباعة محاسبي (CSS فقط) |
| `templates/financials/trial_balance_print.html` | قالب طباعة ميزان المراجعة (HTML + Jinja) |

### استخدام القالب مع Flask

```python
return render_template(
    'financials/trial_balance_print.html',
    report_title='Trial Balance',
    report_title_ar='ميزان المراجعة',
    company_name=company_name,
    tax_number=tax_number,
    address=address,
    logo_url=logo_url,
    show_logo=show_logo,
    generated_at='2026-02-03 14:18',
    asof_date='2026-02-03',
    rows=[{"Code": "1111", "Account": "صندوق رئيسي", "Debit": 0, "Credit": 28220}],
    total_debit=33946.80,
    total_credit=33946.80,
)
```

### توسعة لقوالب أخرى
- **قائمة الدخل (Income Statement)** — نفس الـ partial + جدول (بند، مبلغ).
- **الميزانية العمومية (Balance Sheet)** — نفس الـ partial + أقسام أصول/خصوم/حقوق ملكية.
- **التدفقات النقدية (Cash Flow)** — نفس الـ partial + جدول حركات.
